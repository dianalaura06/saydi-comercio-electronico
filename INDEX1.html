"html" es el leguaje marcado que nos permite indicar la estructura del documento.

*"head" es una de las 2 partes de la estructura del documento en la cual reside la informacion.

*"title" Es el titulo del documento.

*"body background" es un atributo utlizado para especificar una ruta (URL) de la imagen.

*"center" Es utilizado para centrar imagenes y textos

*"font" Es utlizado para modificar el tamaño, la fuete y el color del texto.

*"img src" Representa una imagen en el documento.

*"h1" indica la cabecera mas importante de una pagina.

*"h2","h3",etc.Son indicadore de hipertextos de acuerdo a la importacia del texto.

*"p" Es indicador de un parrafo diferente.

*"td" Es la que representa una celda de datos de una tabla.

*"tr" Es aquella que define un fila dentro de la tabla.

*"a hreF" Se utiliza para indicar la URLa la que apunta el enlace.

*"/" antes de cada etiqueta para indicar el fin o que ha terminado el efecto que queriamos presentar.
